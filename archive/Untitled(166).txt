Today I spent time uploading the source code of my work to GitHub.

This includes WordPress plugins, Upwork Chrome extensions, and game projects. For a long time, I barely used GitHub or shared my code publicly. Most of this work lived on my local machine or old folders.

At some point it hit me that this is years of effort that could quietly disappear. If it is not shared, it helps no one. If it is already solving problems in production, it can probably help someone else as well.

Nothing here is a demo or half built experiment. It is code that has been written, tested, and used in real projects.

Knowledge going to waste helps nobody.
