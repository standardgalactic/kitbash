( sentence

Solve the innermost bubbles first:
( precaution ) → pop → precaution
( default ) → pop → default

Next layer:
( of precaution ) → merge → of-precaution
( the default ) → merge → the-default

Continued collapse:
( paradox of-precaution ) → paradox-of-precaution
( becomes the-default ) → becomes-the-default

Final merge:
( the paradox-of-precaution ) ( dissolves ) ( when trust becomes-the-default )
→ linear text via left-to-right concatenation
