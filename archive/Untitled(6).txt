## What is this repository?

`font-from-grid` is a **public laboratory** for the **Relativistic Scalar-Vector Plenum (RSVP)** framework.  
It contains **two fully-functional, WebGL-accelerated simulators**:

| Simulator | File | Core Idea |
|-----------|------|-----------|
| **Xenobot Swarm Simulator v2.1** | `index.html` | Bio-inspired agents evolve foraging behavior in coupled nutrient (Φ) and entropy (S) fields. |
| **Stars! RSVP Tech-Tree Simulator v2.0** | `stars.html` | 4X-style empires evolve tech priorities, factory deployment, and entropy-aware resource allocation to test **takeoff trajectories**. |

Both tools are **self-contained HTML/JS** files that run in any modern browser. No server, no build step.

## Quick Start

```bash
git clone https://github.com/standardgalactic/font-from-grid.git
cd font-from-grid
```

*Open a simulator:*  

```bash
# Xenobots
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux

# Stars!
open stars.html
```

*Interact:*  
- Tweak sliders (population, λ, mutation, etc.)  
- Press **Start** → watch evolution → **Export CSV**

## Reproducible Experiments

### 1. Install dependencies

```bash
npm install          # puppeteer for batch runs
pip install -r requirements.txt   # pandas, matplotlib
```

### 2. Run a batch (100 generations × N configs)

```bash
node batch_runner.js
```

*Outputs* → `data/*.csv`

### 3. Generate figures

```bash
python analyze.py
```

*Outputs* → `figures/*.pdf`

## Core Features

| Feature | Description |
|---------|-------------|
| **RSVP Fields** | Φ (resource diffusion), v (decision flow), S (entropy trails) |
| **Evolvable Genomes** | 8-D (Xenobot) / 12-D (Stars!) – priorities, rates, thresholds |
| **WebGL Rendering** | `regl.js` – 20 k agents @ 60 FPS |
| **t-SNE Visualization** | Real-time behavioral clustering |
| **CSV Export** | Generation-level + per-agent logs |
| **Headless Automation** | Puppeteer batch runner for Monte-Carlo studies |
