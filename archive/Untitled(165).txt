Here the base is \[ b = \frac{3}{2}, \] so we are working in ``base one and a half.'' Remarkably, every nonnegative integer still has a finite positional representation, but now the digits are restricted to $\{0,1,2\}$: \[ N = \sum_{k=0}^m d_k \left(\frac{3}{2}\right)^k, \qquad d_k \in \{0,1,2\}. \] When we write the number in base $3/2$, the leftmost digit is the coefficient of the highest power of $3/2$, just as in base $10$ or base $2$. Throughout this note, I will call this system the \emph{unidimary} system. \section{The $3\to2$ Exploding Box Rule} The most intuitive way to generate unidimary representations is the ``exploding dots'' or \emph{$3\to2$ box} rule. Imagine an infinite row of boxes extending to the left. We place dots in the rightmost box to represent an integer $N$. The rule is: \begin{quote} Whenever a box contains $3$ dots, we remove those $3$ dots and add $2$ dots to the next box on the left. \end{quote} We repeatedly apply this rule until every box contains at most $2$ dots. At that point, the configuration is stable, and the number of dots in each box (from left to right) gives the digits $d_m d_{m-1} \dots d_0$ of $N$ in base $3/2$. Why does this correspond to base $3/2$? Because the rule \[ 3 \text{ dots in box } k \quad \longrightarrow \quad 2 \text{ dots in box } (k+1) \] encodes the identity \[ 3\cdot\left(\frac{3}{2}\right)^k = 2\cdot\left(\frac{3}{2}\right)^{k+1}. \] In other words, three units of ``value'' in box $k$ are equivalent to two units of value in box $k+1$. When no more explosions are possible, each box contains $0$, $1$, or $2$ dots, giving a valid positional expansion with base $3/2$ and digits in $\{0,1,2\}$. \section{Powers of \texorpdfstring{$\tfrac{3}{2}$}{3/2}} For later calculations it is convenient to list a few powers of $3/2$: \begin{align*} \left(\frac{3}{2}\right)^0 &= 1, \\ \left(\frac{3}{2}\right)^1 &= \frac{3}{2} = 1.5, \\ \left(\frac{3}{2}\right)^2 &= \frac{9}{4} = 2.25, \\ \left(\frac{3}{2}\right)^3 &= \frac{27}{8} = 3.375, \\ \left(\frac{3}{2}\right)^4 &= \frac{81}{16} = 5.0625, \\ \left(\frac{3}{2}\right)^5 &= \frac{243}{32} = 7.59375, \\ \left(\frac{3}{2}\right)^6 &= \frac{729}{64} = 11.390625. \end{align*} These are the ``place values'' of the unidimary system. \section{Example 1: Representing \texorpdfstring{$10$}{10}} \subsection*{Using the $3\to2$ box rule} Start with $10$ dots in the rightmost box. \begin{itemize} \item Take $3$ dots at a time from the rightmost box and convert each triple into $2$ dots one box to the left. After removing $9$ dots (three triples) we have \[ \text{box }0: 1 \text{ dot}, \qquad \text{box }1: 6 \text{ dots}. \] \item Now apply the same rule to box $1$: $6$ dots $\to$ two triples, each giving $2$ dots in box $2$, leaving \[ \text{box }0: 1, \quad \text{box }1: 0, \quad \text{box }2: 4. \] \item Box $2$ still has at least $3$ dots. One more explosion gives \[ \text{box }0: 1, \quad \text{box }1: 0, \quad \text{box }2: 1, \quad \text{box }3: 2. \] Now every box has $0,1,$ or $2$ dots: the process stabilizes. \end{itemize} Reading from left to right, the digits are $2\,1\,0\,1$, so in unidimary notation \[ 10 = (2101)_{3/2}. \] \subsection*{Checking with powers of \texorpdfstring{$3/2$}{3/2}} We can verify this algebraically: \begin{align*} (2101)_{3/2} &= 2\left(\frac{3}{2}\right)^3 + 1\left(\frac{3}{2}\right)^2 + 0\left(\frac{3}{2}\right)^1 + 1\left(\frac{3}{2}\right)^0 \\ &= 2\cdot\frac{27}{8} + 1\cdot\frac{9}{4} + 0\cdot\frac{3}{2} + 1\cdot 1 \\ &= \frac{54}{8} + \frac{18}{8} + 0 + \frac{8}{8} \\ &= \frac{80}{8} \\ &= 10. \end{align*} So the exploding-box construction and the analytic expansion agree. \section{Example 2: Representing \texorpdfstring{$14$}{14}} \subsection*{Using the $3\to2$ box rule} Now start with $14$ dots in the rightmost box. \begin{itemize} \item Three triples of dots explode to box $1$, leaving \[ \text{box }0: 5,\quad \text{box }1: 6. \] \item Explode two triples in box $0$: \[ \text{box }0: -1 + 5 = 2 \text{ (after removing $6$ dots)}, \quad \text{box }1: 6 + 4 = 10. \] So we have \[ \text{box }0: 2,\quad \text{box }1: 10. \] \item Explode triples in box $1$: $10$ dots $\to$ three triples (giving $6$ dots in box $2$) plus one leftover, so \[ \text{box }0: 2,\quad \text{box }1: 1,\quad \text{box }2: 6. \] \item Finally, explode box $2$: $6$ dots give two triples, hence $4$ dots in box $3$: \[ \text{box }0: 2,\quad \text{box }1: 1,\quad \text{box }2: 0,\quad \text{box }3: 4. \] One more explosion in box $3$ leaves \[ \text{box }0: 2,\quad \text{box }1: 1,\quad \text{box }2: 2,\quad \text{box }3: 2. \] Now every box has at most $2$ dots. \end{itemize} Reading from left to right, the digits are $2\,1\,2\,2$, so \[ 14 = (2122)_{3/2}. \] \subsection*{Checking with powers of \texorpdfstring{$3/2$}{3/2}} Again we check analytically: \begin{align*} (2122)_{3/2} &= 2\left(\frac{3}{2}\right)^3 + 1\left(\frac{3}{2}\right)^2 + 2\left(\frac{3}{2}\right)^1 + 2\left(\frac{3}{2}\right)^0 \\ &= 2\cdot\frac{27}{8} + 1\cdot\frac{9}{4} + 2\cdot\frac{3}{2} + 2\cdot 1 \\ &= \frac{54}{8} + \frac{18}{8} + \frac{24}{8} + \frac{16}{8} \\ &= \frac{112}{8} \\ &= 14. \end{align*} \section{Relation to Variational and Field-Theoretic Frameworks} Although unidimary numbers arise from a simple combinatorial rule, they resonate with several themes in my broader theoretical frameworks: \begin{enumerate} \item \textbf{Nonlinear flow with an invariant.} The $3\to2$ rule defines a discrete dynamical system on configurations of dots. The total \emph{value} represented by the configuration is preserved under each explosion, even though the local distribution of dots changes. This mirrors lamphrodynamic flows in RSVP-style field theories, where the fields evolve while preserving an energy-like or information-like quantity. \item \textbf{Multiscale representation.} Each leftward move in the box machine corresponds to multiplication by $3/2$, i.e., a rescaling of the unit. Integer structure is distributed across scales in a way that resembles multiscale tilings and TARTAN-like recursion, where global structure is encoded in how quantities propagate across levels. \item \textbf{Complexity and economy of description.} Integer base $3/2$ expansions use only digits $\{0,1,2\}$, yet they encode integer values exactly. This is a toy example of a complexity-weighted representation: the dynamics push mass leftward until a compact, stable configuration is reached. This is analogous to curvature-minimizing or surprise-minimizing variational flows that compress structure into an economical description. \item \textbf{Bridging discrete and continuous.} Conceptually, the box machine is discrete and combinatorial, while the analytic expansion in powers of $3/2$ is continuous and functional-analytic. The equality between the two pictures echoes the way that discrete cellular automata and continuous PDEs can describe the same underlying semantics in my field-theoretic work. \end{enumerate} In this sense, unidimary numbers and the $3\to2$ box are a small, playful instance of the larger theme: nonlinear local rules giving rise to globally coherent descriptions across scales. \end{document}
