\begin{abstract}
This document specifies the system architecture for Polyxan--RSVP Starspace:
a distributed, hypermedia-based, RSVP-enhanced semantic galaxy platform.
Components include the Content Atom store, Span index, Typed Link engine,
Polycompiler microservice, RSVP Field Engine, Embedding Service, Galaxy View
Renderer, Ship Simulation, g-reset subsystem, and sharded database and cache
layers. UML diagrams and API sketches are provided, along with operational
semantics for user actions and simulations. The specification is suitable
for engineering implementation.
\end{abstract}

\section{High-Level Overview}

The system consists of:

\begin{itemize}
  \item \textbf{Content Layer}: Atoms, Spans, Links.
  \item \textbf{Generation Layer}: Polycompiler, AI generation.
  \item \textbf{Semantic Layer}: Embeddings, RSVP fields.
  \item \textbf{Presentation Layer}: Galaxy starspace renderer.
  \item \textbf{Interaction Layer}: ship movement, projection, reset.
\end{itemize}

\section{Operational Semantics}

Define system state:

\[
\Sigma = (\mathcal{C},\mathbf{Z},\mathbf{X},F,\mathcal{U}).
\]

\subsection{Ship Movement}

\[
\mathbf{x}_u(t+\Delta t)
= \mathbf{x}_u(t) + \Delta t \cdot \mathbf{d}_u
\]

where $\mathbf{d}_u$ is a motion direction determined by controls.

\subsection{Projection}

Projection creates a ghost ship in remote maps.

\subsection{Reset}

Holding \verb|g| for 5s triggers:

\[
\mathbf{Z}' = \mathcal{R}_z(\mathbf{Z},F),\quad
\mathbf{X}' = \mathcal{R}_x(\mathbf{Z}',F,\mathcal{U})
\]

with autoblink constraints.

\section{Scaling and Distribution}

\begin{itemize}
  \item Content graph stored in distributed DB (e.g. CockroachDB).
  \item Embedding service in GPU cluster.
  \item RSVP field engine runs distributed PDE or graph Laplacian solvers.
  \item Starspace renderer is client-side WebGL.
  \item Sharded galaxy caches store precomputed neighborhoods.
\end{itemize}

% =============================================================
\section{Security and Privacy}

\begin{itemize}
  \item All users appear only as anonymous shapes in others' galaxies.
  \item Projection is read-only.
  \item Reset does not reveal identity.
\end{itemize}
